## ✨ Overview

**CodePilot** is a modern, all-in-one developer toolbox built with pure:

- HTML
- CSS
- JavaScript
- Python (minimal backend utilities)

It runs fully offline and includes **50+ developer tools** in one clean interface.

> No frameworks. No tracking. No setup headaches.

---

## 🎯 Why CodePilot Exists

Developers constantly switch between tools like:

- JSON formatters
- Regex testers
- Password generators
- Base64 tools
- Hash calculators

**CodePilot removes that chaos.**

Everything is inside one fast, beautiful interface.
